# 📚 BookNest - Online Book Store

BookNest is a Java Full Stack web application that allows users to browse, search, and purchase books online through a modern and user-friendly interface.

---

## 🚀 Features

### User Module
- User Registration
- User Login & Logout
- Session Management

### Product Module
- View All Books
- Search Books
- Category Filtering
- Product Details Page

### Cart Module
- Add to Cart
- Update Quantity
- Remove Items

### Order Module
- Checkout Process
- Place Orders
- Order Success Page
- View Order History

### UI Features
- Responsive Design
- Modern Dark Theme
- Interactive Product Cards
- Professional Order Confirmation Page
